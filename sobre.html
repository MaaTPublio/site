    > O que eu entendo por desenvolvimento web

    Em desenvolvimento web trabalhamos com a criação de toda parte frontal do site que será entregue para o cliente (no caso quem acessa a página) popularmente chamada de front-end. No front-end se cria toda parte que a pessoa irá ver quando acessar o site, e para isso utilizamos o HTML, que é uma linguagem de marcação de texto, utilizada para organizar o texto, imagens e etc. Para a estilização utilizamos o css que é uma linguagem para toda a parte de estilo da pagina e orientação dos objetos. E por ultimo utilizamos o javascript para realização de todas as funções de botões e codigos em si da página, antes de ir para o back-end.
